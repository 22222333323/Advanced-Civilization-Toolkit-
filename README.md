# WorldBoxMod — Advanced Civilization Toolkit (CCT)

## Описание
WorldBoxMod — мод для WorldBox (v0.51.2), добавляющий набор инструментов для управления юнитами, городами и базовыми игровыми действиями.

Мод позволяет быстрее взаимодействовать с механиками цивилизаций и выполнять некоторые действия вручную через God Powers.

---

## Возможности
- Изменение профессий юнитов (воин, житель, король)
- Спавн зданий через список выбора (две кнопки одна выбора другая Поставить)
- Принудительное создание города на выбранной клетке
- Назначение столицы
- Принудительное добавление юнитов в город
 

---

## Локализация и UI
- Поддержка английского и русского языков
- Интерфейс для выбора зданий (базовый функционал)

---

## Установка
1. Скачайте последнюю версию релиза
2. Поместите `WorldBoxMod.dll` в папку модов игры

---

## Сборка из исходников
1. Откройте `WorldBoxMod.sln` в Visual Studio
2. Соберите проект в режиме Release (.NET Framework 4.7.2)

Перед сборкой необходимо добавить:
- `Assembly-CSharp.dll` из папки игры в `libs/`

---

## Содержимое репозитория
- Исходный код мода (C#)
